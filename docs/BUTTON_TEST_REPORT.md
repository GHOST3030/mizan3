## ملاحظات حول الأزرار في الواجهة الأمامية

بناءً على تحليل الكود المصدري:

| الصفحة | الأزرار الموجودة | التحكم في الظهور |
|--------|-----------------|-----------------|
| المنتجات | إضافة منتج، تعديل، حذف، بحث | `Can permission={PERMISSIONS.MANAGE_PRODUCTS}` |
| العملاء | رصيد افتتاحي، حذف، كشف حساب | `Can permission={PERMISSIONS.MANAGE_CUSTOMERS}` |
| الموردين | إضافة، تعديل، حذف | `Can permission={PERMISSIONS.MANAGE_SUPPLIERS}` |
| المشتريات | إضافة، تعديل، حذف | `Can permission={PERMISSIONS.MANAGE_PURCHASES}` |
| المبيعات | إضافة، تعديل، إلغاء، إرجاع | `Can permission={PERMISSIONS.MANAGE_SALES}` |
| المخزون | تحويل، جرد، تلف | `Can permission={PERMISSIONS.MANAGE_INVENTORY}` |
| المصروفات | إضافة، تعديل، اعتماد، رفض | `Can permission={PERMISSIONS.MANAGE_EXPENSES}` |
| المستخدمين | إضافة، تعديل، حذف | `Can permission={PERMISSIONS.MANAGE_USERS}` |
| الأدوار | إضافة، تعديل، صلاحيات | `Can permission={PERMISSIONS.MANAGE_ROLES}` |

## المشاكل المكتشفة

1. **🔴 متوسط:** زر "Currency Exchange" في الصفحة `/currency-exchange` يستدعي `POST /api/finance/currency-exchange` ولكن لا يوجد GET لعرض المعاملات السابقة.
2. **🔴 متوسط:** صلاحية `PrintTemplatesPage` تستخدم مفتاح صلاحية خاطئ حسب تقرير التدقيق السابق.
3. **🟡 منخفض:** بعض الأزرار مثل "حذف" تستخدم `confirm()` بدلاً من `ConfirmDialog` المخصص.
