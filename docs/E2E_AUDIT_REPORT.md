### 🚨 المشكلة الحرجة

**مصدر cash_register:** إذا اختار المحاسب مصدر `cash_register` بدلاً من `safe`:
```javascript
// يتم خصم الرصيد من الخزينة
await prisma.cashRegister.update({...})
// ثم إنشاء المصروف - خارج transaction!
const result = await prisma.expense.create({...})
```

**المشكلة:** إذا فشل إنشاء المصروف، يتم خصم الرصيد من الخزينة ولكن لا يتم إنشاء المصروف. **المال يضيع!**

| المشكلة | درجة الخطورة |
|---|---|
| `createExpense` مع cash_register بدون transaction (فقدان المال) | **CRITICAL** |
| لا يوجد منع لتعديل المصروف بعد الموافقة | MEDIUM |
