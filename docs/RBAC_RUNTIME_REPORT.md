## المشاكل المكتشفة

1. **🔴 حرج:** معدل الطلبات المسموح به لتسجيل الدخول (5 محاولات/دقيقة) يمنع الاختبار الشامل للأدوار المتعددة.
2. **🔴 متوسط:** بيانات المستخدمين في قاعدة البيانات لا تطابق بيانات البذرة (seed). المستخدمون المفترضون `manager`، `cashier`، `accountant` غير موجودين، وبدلاً منهم `admin2`، `cashir`، `account2` بكلمات مرور غير معروفة.
3. **🔴 متوسط:** لا يوجد مستخدم `super_admin` في قاعدة البيانات الفعلية مما يمنع اختبار الصلاحيات العليا.
4. **🟡 منخفض:** الصلاحيات معرفة في طبقتين (قاعدة بيانات + كود ثابت في `ROLE_PERMISSIONS_MAP`) مما قد يسبب تضارباً.
